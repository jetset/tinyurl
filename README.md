# Readme 

## Install

The app has been built and tested on ruby '2.7.2' and rails 6.1. 

to install if you are using rvm. 

```
  rvm install 2.7.2
  git clone https://github.com/jetset/tinyurl
  cd tinyurl
  bundle 
  rails db:create
  rails generate:keys

```
  
### Run the ruby IDE 

```
 ruby bin/tinyurl-ide.rb
 
```
